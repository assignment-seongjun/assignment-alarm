# 과제 알리미 (Assignment Alarm)

학년/반 단위로 과제를 관리하고 공지를 공유하는 학교용 웹 서비스입니다.

## 핵심 기능

### 1. Google 로그인
### 2. 학생용 과제 화면
### 3. 과제 등록
### 4. 공지 기능
### 5. 알림 기능
### 6. 다크모드

## 관리자 기능

### 1. 전체 학년/반 조회
### 2. 전체 과제 관리
### 3. 전체 공지 관리
### 4. 유저 관리

## SQL 테이블

| 테이블 | 설명 |
|--------|------|
| `users` | 사용자 기본 정보, Google 계정 정보, 학년/반, 알림 여부, 관리자 여부 |
| `assignments` | 과제 제목, 세부사항, 마감일, 대상 학년/반, 작성자 |
| `messages` | 학년 공지/반 공지 내용, 공지 유형, 대상 학년/반, 작성자 |
| `user_assignments` | 학생별 과제 완료 여부를 저장하는 연결 테이블 |

## 페이지별 기능

| 페이지 | 기능 |
|--------|------|
| `login.html` | Google 로그인, 첫 로그인 학년/반 설정 |
| `calendar.html` | 캘린더 기반 과제 조회, 완료 처리, 관리자 범위 필터 |
| `register.html` | 과제 등록, 등록 과제 목록 확인, 관리자 대상 학년/반 선택 |
| `messages.html` | 학년 공지/반 공지 조회, 관리자 공지 작성 |
| `settings.html` | 프로필 수정, 알림 설정, 관리자 과제/공지/유저 관리 |
