# 과제 알리미 (Assignment Alarm)

학년/반 단위로 과제를 관리하고 공지를 공유하는 학교용 웹 서비스입니다.

## 핵심 기능

### 1. Google 로그인
- Google 계정으로만 로그인할 수 있습니다.
- `@bssm.hs.kr` 학교 계정만 허용됩니다.
- 처음 로그인한 사용자는 학년/반을 선택해 계정을 생성합니다.
- Google 프로필 이름과 프로필 이미지를 계정에 반영합니다.

### 2. 학생용 과제 화면
- 자신의 학년/반에 해당하는 과제만 볼 수 있습니다.
- 캘린더 화면에서 과제 마감일 기준으로 과제를 확인할 수 있습니다.
- 과제별 세부사항, 작성자, 마감일을 볼 수 있습니다.
- 본인 과제 완료 여부를 체크할 수 있습니다.
- 완료/미완료 개수를 바로 확인할 수 있습니다.

### 3. 과제 등록
- 일반 사용자는 자기 학년/반 과제만 등록할 수 있습니다.
- 관리자는 원하는 학년/반을 선택해서 과제를 등록할 수 있습니다.
- 과제 제목, 세부사항, 마감일을 함께 저장합니다.
- 등록된 과제는 과제 등록 페이지와 캘린더 화면에서 바로 확인할 수 있습니다.

### 4. 공지 기능
- 공지는 `학년 공지`와 `반 공지`로 나뉩니다.
- 학생은 자기 학년 공지와 자기 반 공지만 볼 수 있습니다.
- 관리자는 전체 학년 공지와 전체 반 공지를 모두 볼 수 있습니다.
- 공지 삭제는 작성자 또는 관리자가 할 수 있습니다.
- 공지 작성은 관리자만 할 수 있습니다.

### 5. 알림 기능
- 상단 알림 버튼에서 최근 과제와 최근 공지를 모아 볼 수 있습니다.
- 알림 목록은 최신순으로 정렬됩니다.
- 새 알림 개수가 배지로 표시됩니다.
- 설정에서 과제 알림 On/Off를 바꿀 수 있습니다.

### 6. 다크모드
- 설정에서 시스템/라이트/다크 모드를 선택할 수 있습니다.
- 선택한 화면 모드는 로그인 화면을 포함한 앱 전체에 적용됩니다.

## 관리자 기능

### 1. 전체 학년/반 조회
- 관리자는 캘린더에서 전체 학년/반 과제를 볼 수 있습니다.
- 드롭다운으로 학년과 반을 골라서 범위를 바꿔 볼 수 있습니다.

### 2. 전체 과제 관리
- 관리자는 모든 학년/반 과제를 확인할 수 있습니다.
- 드롭다운으로 학년/반별 과제를 필터링할 수 있습니다.
- 과제를 삭제할 수 있습니다.
- 과제별 `학생별 완료 현황`을 열어서 누가 완료했고 누가 미완료인지 볼 수 있습니다.

### 3. 전체 공지 관리
- 관리자는 학년 공지와 반 공지를 나눠서 볼 수 있습니다.
- 공지 유형, 학년, 반 기준으로 필터링할 수 있습니다.
- 공지를 삭제할 수 있습니다.

### 4. 유저 관리
- 모든 사용자 목록을 볼 수 있습니다.
- 사용자 이름, 학년, 반을 수정할 수 있습니다.
- 일반 사용자 계정을 삭제할 수 있습니다.

## 관리자 계정 규칙

- Google 이메일의 `@` 앞부분에 `teacher`가 포함되어 있으면 자동으로 관리자 권한이 부여됩니다.
- 추가 관리자 이메일은 환경변수 `ADMIN_GOOGLE_EMAILS`로 지정할 수 있습니다.
- 관리자는 전체 학년/반 과제와 공지를 조회할 수 있습니다.

## SQL 테이블

| 테이블 | 설명 |
|--------|------|
| `users` | 사용자 기본 정보, Google 계정 정보, 학년/반, 알림 여부, 관리자 여부 |
| `assignments` | 과제 제목, 세부사항, 마감일, 대상 학년/반, 작성자 |
| `messages` | 학년 공지/반 공지 내용, 공지 유형, 대상 학년/반, 작성자 |
| `user_assignments` | 학생별 과제 완료 여부를 저장하는 연결 테이블 |

## 페이지별 기능

| 페이지 | 기능 |
|--------|------|
| `login.html` | Google 로그인, 첫 로그인 학년/반 설정 |
| `calendar.html` | 캘린더 기반 과제 조회, 완료 처리, 관리자 범위 필터 |
| `register.html` | 과제 등록, 등록 과제 목록 확인, 관리자 대상 학년/반 선택 |
| `messages.html` | 학년 공지/반 공지 조회, 관리자 공지 작성 |
| `settings.html` | 프로필 수정, 알림 설정, 관리자 과제/공지/유저 관리 |
