# 과제 알리미 (Assignment Alarm)

과제 마감일을 관리하고, 학년/반 단위로 메세지를 공유할 수 있는 웹 애플리케이션입니다.

## 현재 기준 주요 기능

- 이름 + 비밀번호 기반 회원가입 / 로그인
- 회원가입 시 입력한 학년/반 기준으로 과제와 메세지 범위 자동 적용
- 달력 화면에서 과제 확인 및 완료 처리
- 학년 메세지 / 반 메세지 분리 조회
- 최근 과제/메세지를 보는 헤더 알림 버튼
- 과제/메세지 렌더링 시 HTML 이스케이프 처리

## 기술 스택

- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Backend**: Node.js + Express
- **Database**: MySQL 8.0
- **Auth**: JWT + bcrypt
- **Container**: Docker & Docker Compose
- **Deploy**: Railway

## 페이지 구성

| 페이지 | 설명 |
|--------|------|
| `login.html` | 이름 기반 로그인 / 회원가입 |
| `calendar.html` | 과제 캘린더, 완료 처리, 날짜별 과제 확인 |
| `register.html` | 과제 등록 및 내가 등록한 과제 관리 |
| `messages.html` | 학년 메세지 / 반 메세지 작성 및 조회 |
| `settings.html` | 프로필 수정, 알림 설정 |

## 데이터 구조

| 테이블 | 설명 |
|--------|------|
| `users` | 이름, 비밀번호, 학년/반, 프로필, 알림 설정 |
| `assignments` | 과제 제목/내용/마감일, 대상 학년/반 |
| `messages` | 발신자, 내용, 학년/반 대상 메세지 |
| `user_assignments` | 사용자별 과제 완료 상태 |

## 동작 규칙

- 과제 등록은 로그인한 사용자의 학년/반에만 등록됩니다.
- 학년 메세지는 같은 학년 사용자에게만 보입니다.
- 반 메세지는 같은 반 사용자에게만 보입니다.
- 사용자는 본인이 작성한 과제와 메세지만 삭제할 수 있습니다.

## 배포 사용

- Railway에 배포된 주소로 접속합니다.
- 이름 + 비밀번호로 회원가입 또는 로그인 후 사용합니다.
- 로그인 후 달력, 과제 등록, 메세지, 알림 기능을 사용할 수 있습니다.

## 로컬 개발 실행 방법

```bash
docker-compose up -d --build
open http://localhost:3000
```

## Railway 배포 메모

- Web 서비스와 MySQL 서비스를 함께 사용합니다.
- 서버 시작 시 기본 스키마를 자동 생성합니다.
- 현재 인증은 이메일이 아니라 `name` 기준입니다.
- 기존 `users.email` 컬럼이 남아 있으면 서버 시작 시 자동으로 제거되도록 맞춰져 있습니다.
